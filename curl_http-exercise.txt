- What is HTTP?
HTTP stands for "Hypertext Transfer Protocol." It is an application protocol that defines how data is exchanged between a client (such as a web browser) and a server (which hosts web content) over the Internet. HTTP is the foundation of data communication on the World Wide Web.
- What is a URL?
A URL, or Uniform Resource Locator, is a reference or address used to identify a resource on the internet.
- What is DNS?
Domain name service.  It maps A and PTR records.
- What is a query string?
A string passed, most often a question mark
- What are two HTTP verbs and how are they different?
GET and POST.  Get is a retrieve request and post is a write to call.
- What is an HTTP request?
A message sent to a client or server
- What is an HTTP response?
A response to an HTTP request...
- What is an HTTP header? Give a couple examples of request and response headers you have seen.
Specifies information in metadata.  Authorization and Content-Type are examples
- What are the processes that happen when you type “http://somesite.com/some/page.html” into a browser?
Scheme recognition, DNS, connection, HTTP request, server process, HTTP response, rendering
and display.
